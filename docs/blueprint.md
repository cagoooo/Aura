# **App Name**: 靈感發射器

## Core Features:

- 隨機5W1H生成: 使用者可以點擊按鈕來隨機產生5W1H中的每一項（Who, What, When, Where, Why, How）。
- 語法潤飾: 利用AI自動潤飾和調整生成的內容，確保語法正確，並使其更符合台灣的使用習慣。
- 內容一致性檢查: 加入一個AI工具，可以根據當前生成的5W1H元素，判斷是否還需要調整其他項目的內容，以確保整體敘述的一致性。
- 元素鎖定: 在每個5W1H的結果旁邊提供一個「鎖定」按鈕，允許使用者鎖定他們喜歡的元素，避免在重新生成時被更改。
- 結果顯示: 將生成結果以清晰且有組織的格式顯示，方便使用者快速查看和參考。
- 繁體中文支援: 確保所有文字和介面元素都以繁體中文呈現，並符合台灣的語言習慣。

## Style Guidelines:

- 主色：明亮的藍色 (#4285F4)，代表創意和啟發。
- 背景色：淺灰色 (#F5F5F5)，提供舒適的視覺體驗。
- 強調色：橙色 (#FFA726)，用於按鈕和重要提示。
- 選擇清晰易讀的繁體中文字體，確保內容的可讀性。
- 使用簡潔、現代的圖標，每個圖標都應能直觀地代表其功能。
- 介面設計應保持簡潔直觀，重要功能放在顯眼位置。
- 加入輕微的動畫效果，例如按鈕點擊的回饋，增加互動的樂趣。
